# Setup
#### Step 1. Retrieve the Submodules
```
git submodule sync
git submodule update --init --recursive
```

#### Step 2. Choose the Environment within Xcode
Click Scheme Dropdown > **Edit Scheme... > Build Configuration** Dropdown > Preferred Environment
